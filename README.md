# 码匠论坛

码匠论坛是一个功能基本全面的内部交流使用论坛平台，提供发布问题、个人资料展示、评论、回复、搜索、接收消息通知等基本功能。后端采用了 SpringBoot + MyBatis + MySQL进行构建，前端采用了 Thymeleaf + Layui 进行展示。界面风格简洁，尚未进行深度美化。如有 BUG 请尽快联系作者提交问题，不胜感激！
